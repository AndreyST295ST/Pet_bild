# Анализ объявлений о пропаже и находке животных

Данный проект предназначен для анализа успешности объявлений о пропаже и находке животных с последующей визуализацией результатов.

## Структура проекта

Pet_build/
├── Dockerfile # Файл для сборки Docker-образа
├── requirements.txt # Список зависимостей Python
├── main.py # Главный исполняемый файл
├── src/ # Исходный код модулей
│ ├── init.py
│ ├── step*.*.py
│ └── deps.py
├── results/ # Папка для сохранения результатов (создается автоматически)
└── README.md # Документация проекта
# Сборка Docker-образа

sudo docker build -t build_v1.0 .

Если ошибка при сборке, то попробуйте прописать эту команду:
sudo docker build --no-cache -t build_v1.0 .
№ Запуск образа
sudo docker run -it build_v1.0:latest
